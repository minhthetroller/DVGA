About DVGA

Damn Vulnerable Go App (DVGA) is an educational web application designed to teach web security concepts.

Vulnerability Categories

DVGA covers the top 5 categories from the OWASP Top 10 (2021):

  1. A01: Broken Access Control — IDOR, Path Traversal, Privilege Escalation
  2. A02: Cryptographic Failures — Data Exposure, Weak Password Storage
  3. A03: Injection — SQL Injection, Blind SQL Injection, Command Injection, XSS (Reflected), XSS (Stored)
  4. A04: Insecure Design — Password Reset, Login Brute Force
  5. A05: Security Misconfiguration — Debug Info, Security Headers

Tech Stack

  • Go (with chi router)
  • GORM + SQLite
  • Docker

Disclaimer

This application is intentionally vulnerable. It should only be used in controlled environments for educational purposes. Do not deploy on any public-facing server.
